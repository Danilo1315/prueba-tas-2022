# prueba-tas-2022
Prueba webApp

* Angular 7.2.0
* Lenguaje: TypeScript

Si tienes instalado Nodejs realiza la siguiente instalacion, de lo contrario
puede descargar en https://nodejs.org/es/download/

Install Project
********************************
npm install -g @angular/cli@7
ng g p Pipes/filter
ng add ngx-bootstrap



Execute command in console.
********************************
ng serve
